# 医生投放评价系统开发文档

## 项目概述

本项目是为药厂品牌部开发的医生投放评价系统，用于评估在抖音、视频号、快手等平台投放医生的价值和效果。系统通过大数据分析和人工评分相结合的方式，为投放决策提供科学依据。

## 技术架构

- **后端**: Rust + Actix Web
- **前端**: HTML/CSS/JavaScript
- **数据库**: SQLite/PostgreSQL
- **架构模式**: RESTful API + 前后端分离
